# ТЗ на иконки — Ultimate Snake Attack (лобби + магазин)

## Как это работает технически

Каждая иконка в интерфейсе сейчас — emoji-заглушка. Код уже настроен
подхватывать готовый арт **автоматически**: как только файл с указанным
именем появляется в этой папке (`/public/img/icons/`), он подменяет собой
заглушку без единой правки HTML/CSS/JS. Если файла нет — просто остаётся
emoji, никаких ошибок и «битых картинок» не будет.

**Всё, что нужно сделать художнику — положить файл с точным именем из
таблицы ниже в `/public/img/icons/`.**

## Общие требования

| Параметр | Значение |
|---|---|
| Формат | PNG с прозрачным фоном (предпочтительно) или SVG |
| Размер холста | 128×128 px (экспорт под retina, в интерфейсе покажется мельче) |
| Безопасное поле | оставляй ~12 px отступа от края холста — иконка не должна упираться в границы |
| Стиль обводки/заливки | плоский/флэт-дизайн с тонким неоновым свечением по краю — под стилистику остального UI (см. ниже референс палитры) |
| Насыщенность | иконка должна хорошо читаться и на тёмном фоне (#030406), и внутри полупрозрачной карточки кнопки |
| Вес файла | старайся уложиться в ≤80 KB на иконку (для PNG — сожми через TinyPNG/аналог) |

### Референс палитры (под неё стилизуем свечение/акценты в иконках)

- Основной неон: `#3de88a` (зелёный)
- Синий акцент: `#5ba4f5`
- Жёлтый акцент (монеты/сундук): `#ffd166`
- Пурпурный акцент: `#c77dff`
- Красный акцент (уведомления/бейджи): `#f25f4c`
- Фон карточек кнопок: тёмный `#080c12` с лёгким блюром

Иконка не обязана быть строго зелёной — но должна иметь достаточный
контраст/свечение под один из акцентов, чтобы не «терялась» на тёмном фоне.

### Состояния

Сейчас система поддерживает **один статичный файл на иконку** — эффект
ховера/нажатия уже даёт сама кнопка (подсветка рамки, лёгкий подъём).
Отдельные hover/active-варианты иконок пока не нужны — если что, добавим
это отдельным этапом.

## Куда класть файлы

```
/public/img/icons/
  brand-mark.png
  chest.png
  play.png
  shop.png
  battlepass.png
  friends.png
  leaderboard.png
  settings.png
  play-online.png
  play-room.png
```

Названия — строго как в таблицах (регистр важен, расширение `.png`).
Можно сдавать не всё сразу: интерфейс одинаково корректно работает
и с полным набором иконок, и с частичным (недостающие останутся emoji).
